# C4 Model (mínimo)

## C1 — Contexto
MindEase ajuda usuários a reduzir fricção cognitiva (TDAH/TEA/Dislexia/Burnout).

## C2 — Containers
- Web App (host + remotes)
- Backend API (preferências, tarefas, auth)
- Storage local (cache/offline)
- Mobile (futuro)

## C3 — Componentes (Web)
- Preferences (painel cognitivo)
- Tasks (kanban + checklist + transição)
- FocusTimer (pomodoro adaptado)
- Profile (wizard + recomendações)
