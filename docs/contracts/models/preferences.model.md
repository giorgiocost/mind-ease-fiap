# Preferences Model (Acessibilidade Cognitiva)

- complexity: simple | medium | full
- focusMode: boolean
- readingMode: summary | detailed
- contrast: default | high
- fontSize: number (ex.: 14..22)
- lineSpacing: number (ex.: 1.2..2.0)
- animations: off | reduced | normal
- cognitiveAlerts:
  - thresholdsMinutes: number[] (ex.: [25,45,60])
  - snoozeMinutesDefault?: number
