# Screenshots

Place application screenshots here for use in the README and documentation.

## Required screenshots

| File | Page | How to capture |
|------|------|----------------|
| `dashboard.png` | `/dashboard` | `npm start` → navigate to `/dashboard` |
| `kanban.png` | `/tasks` | `npm start` → navigate to `/tasks` |
| `pomodoro.png` | `/tasks/pomodoro` | `npm start` → navigate to `/tasks/pomodoro` |
| `preferences.png` | `/dashboard/preferences` | `npm start` → navigate to `/dashboard/preferences` |

## Guidelines

- Browser zoom: **100%**
- Window size: **1440 × 900** (standard laptop)
- Use Chrome / Chromium for consistency
- Capture in **light mode** (default theme)
- Format: **PNG**, max 1 MB each
