# Guia — Acessibilidade Cognitiva (MindEase)

## Regras de UI (obrigatórias)
- Densidade controlada, hierarquia clara, espaçamento consistente
- Texto curto e acionável
- Resumo por padrão; detalhamento sob demanda
- Modo foco sempre disponível e evidente
- Animações controláveis (off/reduced/normal)
- Consistência de navegação e feedback

- [ ] foco visível
