# PR Checklist (para IA e humanos)

## Arquitetura
- [ ] respeita camadas (presentation/app/domain/infra)
- [ ] não importou infra direto na UI
- [ ] boundaries não foram quebradas

## Produto (cognitivo)
- [ ] texto curto e objetivo
- [ ] suporte a resumo/detalhado quando aplicável
- [ ] modo foco não foi afetado
- [ ] animações respeitam preferências

## Qualidade
- [ ] unit tests adicionados/ajustados
- [ ] sem logs soltos
- [ ] docs/ADR atualizados quando necessário
