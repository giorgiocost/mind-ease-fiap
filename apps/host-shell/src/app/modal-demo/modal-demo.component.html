Accessibility Features

  • ✅ Focus trap - Tab/Shift+Tab contidos dentro do modal
  • ✅ Escape key - Fecha o modal (se closeOnEscape=true)
  • ✅ Backdrop click - Fecha ao clicar fora (se closeOnBackdrop=true)
  • ✅ Body scroll lock - Previne scroll da página ao fundo
  • ✅ Focus restoration - Restaura foco ao elemento anterior
  • ✅ ARIA attributes - role="dialog", aria-modal, aria-labelledby
  • ✅ Keyboard navigation - Navegação completa por teclado
